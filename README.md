# Netflix-data-cleaning
Netflix Exploratory Data Analysis (EDA)

Project Overview:

This project explores the Netflix catalog to uncover trends in content types, target demographics, and geographic production using data visualization.

Using Python and data analysis techniques, we perform exploratory data analysis (EDA) to uncover patterns and insights from the dataset.

Link to Project Page: https://github.com/Roshanreddy03/Netflix-data-cleaning

Dataset

Source: Kaggle Netflix Data Set

File used: netflix_titles.csv

---------------------------------------------------------------------------------------------------------------------------------------------------------

Technologies Used:

- Python
  
- Pandas (data cleaning)
  
- Matplotlib (static graphs)
  
- Jupyter Notebook (analysis)

---------------------------------------------------------------------------------------------------------------------------------------------------------

Data Exploration:

- Cleaning: Handled missing values in director/cast/country columns.

- Engineering: Extracted years from dates to analyze addition trends.

- Visualization: Built foundational charts to map distributions and relationships.

---------------------------------------------------------------------------------------------------------------------------------------------------------

Key Insights:

- Format: Movies significantly outnumber TV shows on the platform.

- Growth: The rate of new content added has surged in recent years.

- Geography: The United States is the top content producer, followed by India.

- Audience: The library heavily targets mature audiences (TV-MA, TV-14).

---------------------------------------------------------------------------------------------------------------------------------------------------------

Visualizations:

The notebook includes:

- Core Data Cleaning
  - Handling Missing Values
  - Fixing the Mixed-Type duration Column
  - Parsing the Date Column

- Netflix graphs
  - Bar Plot of Movies vs. TV Shows
  - Histogram of Release Years
  - Horizontal Bar Plot of Top 5 Countries

---------------------------------------------------------------------------------------------------------------------------------------------------------

Conclusion:

Netflix's strategy relies heavily on US-produced movies targeted at mature audiences, though it maintains a diverse global catalog.
